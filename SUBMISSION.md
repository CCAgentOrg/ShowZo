## 🚀 What It Does

ShowZo turns any URL + natural-language description into a produced walkthrough video — autonomously.

Instead of manually recording screencasts, writing narration scripts, and editing video, you type:

> "Show me how to search on example.com and read the first result"

…and ShowZo does the rest:

1. **Plans** the walkthrough using AI (via Zo's `/ask` API)
2. **Opens** a real browser and executes each step (click, type, scroll, hover)
3. **Captures** screenshots at every step
4. **Generates** voice narration (Edge TTS)
5. **Assembles** everything into an MP4 video with transitions

## 🧩 Architecture

```
User Input (URL + Scenario)
         │
         ▼
    ┌─────────────┐
    │  LLM Plan   │  ← Zo /ask generates step-by-step plan
    │  Generator  │
    └──────┬──────┘
           │
    ┌──────▼──────┐
    │  Browser    │  ← agent-browser CLI executes steps
    │  Recorder   │     real browser, real interactions
    └──────┬──────┘
           │
    ┌──────▼──────┐
    │  Narration  │  ← Edge TTS per step
    │  Engine     │
    └──────┬──────┘
           │
    ┌──────▼──────┐
    │  Video      │  ← ffmpeg assembles screenshots
    │  Assembler  │     + narration → MP4
    └─────────────┘
```

**Tech stack:** Bun + Hono + React + Vite + agent-browser + ffmpeg + Edge TTS

## 🛠 Implementation Details

### Pipeline Modules (`src/showzo/pipeline/`)

| Module | Lines | What it does |
|--------|-------|-------------|
| `types.ts` | 72 | Type definitions (Plan, Session, Step, etc.) |
| `runner.ts` | 295 | Agent-browser execution engine |
| `record.ts` | 255 | Narration generation + video assembly |

### API Server (`server.ts`)

| Route | Method | Purpose |
|-------|--------|---------|
| `/api/plan` | POST | Generate walkthrough plan from URL + scenario |
| `/api/record` | POST | Start recording a plan |
| `/api/session/:id` | GET | Poll recording progress |
| `/api/sessions` | GET | List all sessions |
| `/api/health` | GET | Health check |

### Web UI (`src/showzo/`)

- **Input screen**: URL + scenario form with validation
- **Plan review**: See generated steps, approve before recording
- **Recording progress**: Live step-by-step status with timestamps
- **Video preview**: Download completed walkthrough
